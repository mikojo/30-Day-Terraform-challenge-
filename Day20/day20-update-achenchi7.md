- **Date and Time:** 02/01/2025 


Goal: Understand the seven steps of a typical workflow for taking application code from development to production:

These steps are:
- Use version control.
- Run the code locally.
- Make code changes.
- Submit changes for review.
- Run automated tests.
- Merge and release.
- Deploy.
